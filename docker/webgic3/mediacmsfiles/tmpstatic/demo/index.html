About Us

OpenPress is a web platform which allows verified and independent journalists to publish media available for everyone.

OpenPress goal is to allow users to watch media from independent sources, this way fighting the spreading of misinformation and fake news.

With its tag system users can search for a specific topic. They can create playlists and follow specific journalists.
